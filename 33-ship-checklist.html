都不能跳。

1 · UUID 唯一性

games.json 新 entry 有 UUID 且不重复（fork 必踩坑）

2 · meta.json

title（英文！）+ cover_url 齐全，海报放 public/

3 · build 通过

npm run build 全过，无 TS error

4 · 反模式扫描

onPointerDown 不混 mouse+touch · daily-lock 不 OR stats · crossOrigin 不加

5 · Demo 模式自测

浏览器外打开 URL 看 phantom 体验是否正常

6 · 身份层验证

关掉头像/名字，游戏还成立吗？成立 = 身份只是装饰

7 · 部署验证

push → GH Actions → curl bundle URL 看新 class 在不在

8 · 真机自测 5 次

手机划到 feed 里测 5 次（cold start / 弱网 / 横屏）

讲师备忘 · 按 S 关闭

最后一条最容易被忽略——用户实玩前不要算"上线成功"。 讲完后让同学拿手机现场试一个游戏，体会"feed 里看到 vs 单页打开"的区别。
